🤖 KIgeezer Pipeline - TikTok Integration

Automated video creation and publishing to TikTok via Content Posting API

🔐 Login Kit - TikTok Authentication

1 User clicks "Login with TikTok"

✅ User authenticated via TikTok OAuth 2.0 PKCE
Scopes granted: user.info.profile user.info.stats video.upload video.publish

🎬 Content Posting API - Video Upload

2 Pipeline generates video and uploads to TikTok

🎥 Generated Video Preview
B-Roll clip (5-15s) + Voiceover + Text overlay
✅ Video uploaded via Content Posting API
POST /v2/post/publish/video/upload/

📤 Content Posting API - Video Publish

3 Human review → Publish to TikTok

✅ Video published to TikTok
POST /v2/post/publish/content/query/
Status: PUBLISH_COMPLETE

📊 API Flow Summary

1. User → "Login with TikTok" → OAuth 2.0 PKCE Auth
2. Pipeline → Generate B-Roll + Voiceover + Text
3. Pipeline → Upload video via Content Posting API
4. Human  → Review and approve
5. Pipeline → Publish video to TikTok

Domain: kigeezer.com | All content human-reviewed before publishing
